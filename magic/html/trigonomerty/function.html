Trigonometric Functions

Trigonometric functions, also known as circular functions or angle functions, are real functions that relate an angle of a right-angled triangle to ratios of two side lengths.

Sine (sin)

The sine function returns the ratio of the length of the opposite side to the hypotenuse.

sin ( θ ) = opposite hypotenuse

Cosine (cos)

The cosine function returns the ratio of the length of the adjacent side to the hypotenuse.

cos ( θ ) = adjacent hypotenuse

Tangent (tan)

The tangent function returns the ratio of the length of the opposite side to the adjacent side.

tan ( θ ) = opposite adjacent

Cosecant (cosec)

The cosecant function is the reciprocal of the sine function.

cosec ( θ ) = 1 sin ( θ )

Secant (sec)

The secant function is the reciprocal of the cosine function.

sec ( θ ) = 1 cos ( θ )

Cotangent (cot)

The cotangent function is the reciprocal of the tangent function.

cot ( θ ) = 1 tan ( θ )
