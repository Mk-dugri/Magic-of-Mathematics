Quadratic Equations

Introduction to Quadratic Equations

Quadratic equations are polynomial equations of the form:

ax2 + bx + c = 0

Factoring Method

Learn how to solve quadratic equations by factoring:

x2 + 5x + 6 = 0

(x + 2)(x + 3) = 0

x = -2, x = -3

Completing the Square

Understand how to complete the square to solve quadratic equations:

x2 + 6x + 9 = 0

(x + 3)2 = 0

x = -3

Quadratic Formula

Explore the quadratic formula:

x = \(\frac{-b \pm \sqrt{b^2 - 4ac}}{2a}\)

Example: x2 - 4x - 5 = 0

x = \(\frac{4 \pm \sqrt{16 + 20}}{2}\)

Graphing Quadratic Equations

Quadratic equations can be graphed on a coordinate plane. The graph is a parabola that opens upwards or downwards.

Example: y = x2 - 4x - 5

Applications of Quadratic Equations

Quadratic equations are used in various real-world applications such as projectile motion, area problems, and economics.
