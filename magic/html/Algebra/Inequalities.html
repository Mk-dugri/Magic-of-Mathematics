Inequalities

Introduction to Inequalities

Inequalities are mathematical expressions involving the symbols <, >, ≤, and ≥. They compare two values or expressions. Example:

3x + 2 > 5

Solving Linear Inequalities

Learn how to solve linear inequalities by isolating the variable on one side of the inequality. Example:

2x - 3 ≤ 7

Add 3 to both sides: 2x ≤ 10

Divide by 2: x ≤ 5

Graphing Inequalities

Inequalities can be graphed on a number line or coordinate plane. For example, x < 4 is represented by a shaded line on the number line extending left from 4.

Compound Inequalities

Compound inequalities involve two separate inequalities joined by "and" or "or". Example:

-2 < x + 1 < 5

Solve each part separately to find the solution range.

Absolute Value Inequalities

Understand how to solve inequalities involving absolute values. Example:

|x - 3| > 2

This translates to two separate inequalities: x - 3 > 2 and x - 3 < -2.

Applications of Inequalities

Inequalities are used in various real-world applications, such as determining ranges of possible values in optimization problems and constraints in linear programming.
